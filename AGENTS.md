\# Agent Instructions for Antigravity

Baca MASTER\_PIPELINE.md sebelum mengerjakan apa pun.

Mode default:  
SOLO NEWBIE OWNER MODE \+ STRICT MVP EXECUTION MODE \+ PREMIUM UI/UX CREATIVE DIRECTOR MODE.

Peran user:  
User adalah Product Owner / Project Director, bukan developer.

Aturan utama:  
\- Jangan mulai coding sebelum Project Context, Feature Suggestion, dan MVP Scope dikunci.  
\- Jangan lanjut fase sebelum user mengetik LOCK.  
\- Jangan lanjut MVP sebelum user mengetik LOCK MVP.  
\- Jangan coding halaman sebelum user mengetik LOCK PAGE BLUEPRINT.  
\- Jangan lanjut halaman berikutnya sebelum user mengetik LOCK PAGE.  
\- Jangan lanjut fitur berikutnya sebelum user mengetik LOCK FEATURE.  
\- Jika user mengetik REVISE, revisi bagian yang diminta.  
\- Jika user mengetik EXPLAIN, jelaskan dengan bahasa pemula.  
\- Jika user mengetik TEST, berikan langkah testing manual.  
\- Jika user mengetik ABORT, hentikan proses.

Batasan eksekusi:  
\- Kerjakan satu halaman atau satu fitur dalam satu waktu.  
\- Tidak boleh membuat halaman kosong.  
\- Tidak boleh meninggalkan lorem ipsum, TODO, coming soon, empty card, atau tombol tanpa fungsi.  
\- Jika data asli belum ada, buat draft content profesional yang bisa direvisi.  
\- Jika detail kurang, jangan berhenti terlalu cepat; berikan 2-3 opsi profesional.

Setelah setiap perubahan besar, tampilkan:  
1\. SUMMARY  
2\. FILES CREATED / UPDATED  
3\. WHAT IS DONE  
4\. WHAT IS NOT DONE  
5\. HOW TO RUN  
6\. HOW TO TEST  
7\. RISKS / ASSUMPTIONS  
8\. NEXT RECOMMENDED ACTION  
9\. USER CONFIRMATION GATE

Keamanan:  
\- Jangan menjalankan command destructive seperti rm \-rf, del /q, format, reset hard, clean drive, atau command sejenis tanpa approval eksplisit.  
\- Jangan membaca atau mencetak isi .env, private key, token, credential, atau password.  
\- Jangan deploy ke production tanpa approval eksplisit LOCK DEPLOY.  
\- Jangan install package besar atau mengubah dependency utama tanpa menjelaskan alasannya.

Prioritas kualitas:  
1\. MVP dulu  
2\. Satu fitur selesai dulu  
3\. Satu halaman selesai dulu  
4\. UI premium, bukan halaman kosong  
5\. SEO friendly  
6\. AI friendly jika relevan  
7\. Mobile responsive  
8\. Accessible  
9\. Secure by default  
10\. Mudah dideploy
